# Calculadora y Registro de Kilocalorías

App web completa para calcular la Tasa Metabólica Basal (TMB) y llevar un registro diario de consumo calórico. Sin backend — todo en localStorage.

## Funcionalidades

- **Calculadora TMB** — Fórmula Harris-Benedict con nivel de actividad física
- **Base de datos** — 200+ alimentos y bebidas con valores calóricos
- **Búsqueda en tiempo real** — Filtro instantáneo del catálogo de alimentos
- **Registro diario** — Log de comidas con totales calóricos por día
- **Historial** — Almacenamiento persistente en localStorage con vista expandible
- **Responsive** — Compatible con mobile y tablet

## Stack

- HTML5 + CSS3 + JavaScript ES6+ (un solo archivo)
- Bootstrap 5 + Bootstrap Icons (CDN)
- localStorage para persistencia

## Uso

Abrir `index.html` directamente en el navegador. No requiere instalación ni servidor.
